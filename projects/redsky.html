redsky

redsky is a simple viewer for Bluesky that requires no JavaScript, inspired by invidious and nitter.

features

  • extremely lightweight, you are given no scripts at all
  • works great on anything that can browse the web (maybe)
  • good for utmost simplicity, possibly circumventing censorship, scraping, or just for fun

why?

the bluesky web app will not load at all unless you havee javascript enabled. this isn't a problem for most people, but if you want to use bluesky on a device that does not support javascript, or just like simple things, you might like redsky.

check it out!
