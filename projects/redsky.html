redsky

status: KILLED
inception: June 2025
killed: November 2025

Redsky was a simple viewer for Bluesky that requires no JavaScript, inspired by Invidious and Nitter.

features

  • Extremely lightweight, you are given no scripts at all
  • Works great on anything that can browse the web (maybe)
  • Good for utmost simplicity, possibly circumventing censorship, scraping, or just for fun
visit redsky

why?

The Bluesky web app will not load at all unless you have JavaScript enabled. This isn't a problem for most people, but if you want to use Bluesky on a device that does not support javascript, or just like simple things, you might like Redsky.

shutdown

On September 25 Redsky was shut down because it became a weaponized firehose for aggressive scrapers, which I did not want to be a part of.
Read more at this post.

reboot

On October 16 I turned it back on with heavy rate limiting and bot detection in place. No idea if I can call it "active" again. Read more at this post.

second shutdown

On November 2 after my attempts to curb abuse failed, I decided to pull the plug again; permanently this time. Read more at this post.
